# Financial Planner 2026
Простое и надежное приложение для управления личными финансами.

## Возможности
- Добавление операций (доходы/расходы).
- Изменение уже добавленных операций (доходы/расходы).
- Валидация данных через регулярные выражения.
- Установка лимита или Бюджета на расходы с визуализацией в случае превышения лимита.
- Сортировка списка покупок по все полям.
- Фильтрация списка по категориям и диапазону дат.
- Кнопка сброса всех фильтров.
- Хранение истории в CSV файле.
- Экспорт списка в CSV файл.
- Аналитика с помощью Pandas.
- Визуализация отчетов: "Круговая диаграмма", "Помесячный график расходов"(Matplotlib).

## Как запустить
1. Установите зависимости: `pip install pandas matplotlib`
2. Запустите программу: `python main.py`
3. Для тестов: `python tests.py`
